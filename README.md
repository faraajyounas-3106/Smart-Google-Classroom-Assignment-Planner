# Smart-Google-Classroom-Assignment-Planner

Smart GCR Planner is a browser extension designed to help students organize and plan their assignments efficiently while keeping deadlines in view. It automatically helps break down tasks and assists in better time management for academic work.

---

## 🚀 Features

- 📅 Add and manage assignments easily  
- ⏰ Tracks deadlines and prioritizes tasks  
- 🧠 Smart planning based on due dates  
- 📊 Helps students distribute workload effectively  
- 🔔 Deadline reminders (if implemented)  
- 🧩 Simple and lightweight interface  

---

## 🎯 Purpose

Students often struggle with managing multiple assignments at once. Smart GCR Planner solves this by providing a structured way to plan tasks and avoid last-minute stress.

- HTML

- Chrome Extension APIs (if applicable)

## 📦 Installation

1. Clone or download this repository  
2. Open Chrome and go to `chrome://extensions/`  
3. Enable **Developer Mode**  
4. Click **Load Unpacked**  
5. Select the project folder  
6. The extension will now be installed  

---

## 🧑‍💻 How to Use

1. Open the Smart GCR Planner extension  
2. Add your assignment details (title, subject, deadline)  
3. The planner will organize tasks based on urgency  
4. Track progress and complete tasks before deadlines
